## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Yamuna-97/atm_application.git
cd atm_application
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## 🚀 How It Works

1. User creates an account or logs in
2. Authentication verifies credentials
3. User accesses ATM dashboard
4. Deposit or withdraw money
5. Account balance updates instantly
6. Transaction records are stored in MongoDB
